# 04 — 错误等级规范

## P0 — 立即停止，禁止发布

可能误发/错发/错商品/错账号。

- 商品挂错
- 账号错位
- live 模式下商品未校验成功
- live 模式下图片/标题/正文为空
- publish_mode 不明确却准备点击发布
- 客户 A 的素材写入客户 B 的任务目录

**动作：** `stop_task` — 停止整个任务，所有未执行的账号标记为 skipped

## P1 — 当前账号失败，进入下个账号

当前账号无法继续。

- 当前账号未登录
- 账号风控
- 页面结构变化
- 商品搜不到
- 生成图失败
- 文案多次不合格

**动作：** `stop_account` — 停止当前账号，记录 bugs.json，进入下个账号

## P2 — 可重试问题

- 页面加载慢
- 按钮短暂不可点击
- 图片上传失败
- 网络请求失败
- 下载失败

**动作：** `retry` — 按配置重试，重试耗尽后降级为 P1

## P3 — 非关键问题

- 非关键截图失败
- summary.md 生成失败
- 额外统计信息缺失

**动作：** `log_only` — 记录但不中断
